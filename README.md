# [oshayjackson.github.io](https://oshayjackson.github.io)

Personal developer portfolio built with **React**, **TypeScript**, and **Vite**, statically deployed to **GitHub Pages**.

## Tech Stack

- React
- TypeScript
- Vite
- GitHub Pages
- GitHub Actions

## Local Development

```sh
npm install
npm run dev
```

## Build & Preview

```sh
npm run build
npm run preview
```
